# AIO Conquer 2026 — Project

Project competition song hành cùng AIO2026 (Module 01→10 ↔ Project 01→10).

**Vai trò của mình:** **Tech Leader** (trưởng nhóm kỹ thuật) — định hướng, điều phối và đảm bảo tiến độ của nhóm; cầu nối giữa thành viên và Ban Tổ Chức (BTC).

## Trách nhiệm chính

- **Rã task:** chiến lược ngắn hạn (theo tuần) + dài hạn (theo module), mỗi task có mục tiêu đầu ra + deadline rõ ràng.
- **Nhịp họp:** chốt một buổi họp cố định mỗi tuần, luôn diễn ra để review (dù task xong hay chưa).
- **Tracking giữa tuần:** hỏi han, gỡ vướng cho thành viên trước khi tới deadline.
- **Merge output:** tổng kết, đóng gói, hợp nhất task của các thành viên → sản phẩm hoàn chỉnh trước deadline.
- **Quy trình task:** giao (assign) → báo cáo (report) → phản hồi (feedback) → cải tiến (improve).

## Công cụ & kỹ năng

- **Kỹ năng:** task decomposition, time estimation, quản lý deadline.
- **Công cụ:** Jira (theo dõi task) + Agile/Scrum (giản lược cho nhóm nhỏ).
