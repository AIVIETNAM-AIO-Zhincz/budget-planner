# Naming Conventions

- `snake_case` — hàm, biến, module, file Python.
- `PascalCase` — class, Pydantic model, SQLAlchemy model.
- `UPPER_SNAKE` — hằng số.
- Tên rõ nghĩa, không viết tắt mơ hồ; tên hàm là động từ (`calculate_total`), tên biến là danh từ.
- File code mỗi buổi học: `M{NN}-.../W{NN}/{thứ tự}_{chủ đề}/`.
- Bảng database & cột: `snake_case`, số nhiều cho bảng (`transactions`), khoá ngoại `{entity}_id`.
